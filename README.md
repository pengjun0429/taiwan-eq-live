# 📍 台灣地震即時監測儀表板 (FJM EQ Monitor)


這是一個專為台灣設計的**輕量化地震監測網頁**。本專案捨棄繁雜的地圖，專注於提供最直觀的文字數據，具備自動命名的智慧功能，並針對**高雄鳳山地區**進行距離運算。

---

## 🚀 核心亮點 (Key Features)

* **⚡ 今日最新震報**：
    * 網頁頂部區域鎖定顯示「當天」發生的最新地震。
    * 若今日平安無事，系統會自動切換為綠色「今日平安」狀態。
* **🏷️ 自動化智慧命名**：
    * **分頁標籤**：自動將瀏覽器標籤更名為 `【規模】地點`，不用點進網頁即可掌握災情。
    * **網頁大標**：中央標題隨最新地震地點自動更新（例如：📍 花蓮縣近海）。
* **📏 鳳山專屬距離計算**：
    * 利用球面三角法公式 (Haversine Formula) 即時計算震央與**鳳山區**的直線距離。
* **📅 三日歷史深度過濾**：
    * 自動抓取並分類「今天、昨天、前天」的所有顯著地震，按日期整齊排列。
* **🚨 強震視覺動畫**：
    * 規模 $\ge$ 4.0 的地震自動套用**紅色邊框警示**與**震動動畫特效**。

---

## 🛠️ 技術細節 (Technical Details)

本專案採用純前端架構，無需後端資料庫，數據即時且穩定：

* **數據來源**：[中央氣象署 (CWA)](https://opendata.cwa.gov.tw/) 開放資料平台 API。
* **前端技術**：HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+)。
* **核心算法**：
    $$d = 2R \cdot \arcsin\left(\sqrt{\sin^2\left(\frac{\Delta\phi}{2}\right) + \cos\phi_1\cos\phi_2\sin^2\left(\frac{\Delta\lambda}{2}\right)}\right)$$
    *(註：$R = 6371$ km 為地球平均半徑)*
